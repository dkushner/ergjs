ergjs

Simplified parallelism in the browser through Web Workers.

Download .zip Download .tar.gz View on GitHub

What is it?

ergjs is a library that makes it easy to interact with Web Workers in your site. It removes the hassle of integrating Web Workers for specific applications by managing individual, or groups of web workers as general "task do-ers."

Getting Started To use ergjs in your project, simply install via bower using:

bower install ergjs

And include the relevant script tag in your document:

<script src="/bower_components/ergjs/lib/erg.js"></script>
